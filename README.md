# Currency Exchange Tracker Web App

A simple web application that allows users to convert currencies using real-time exchange rates from the **ExchangeRate-API**.

## Features

- Real-time currency conversion
- Supports multiple currencies
- Displays up-to-date exchange rates
- Supports Historical Data of Exchange Rate
- Simple and user-friendly interface 

## Technologies Used

- HTML
- CSS
- JavaScript
- ExchangeRate-API

## Description

This project is designed to help users quickly convert one currency to another using live exchange rate data. It provides a fast, simple, and responsive interface for accurate currency conversion.
